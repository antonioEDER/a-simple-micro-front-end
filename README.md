# a-simple-micro-front-end
Um projeto desenvolvido com Vue Microfrontends

```
Um projeto desenvolvido com Vue Microfrontends

Nele vamos explorar ainda:
- single-spa
- VueJs

```

<h2 align="center">Install</h2>

Install create-single-spa:
```bash
yarn add create-single-spa
```

Generation project:
```bash
create-single-spa
```
Start project:
```bash
yarn start
```
